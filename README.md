# JVM Compiler – C (Flex & Bison)

A compiler for a custom language that generates JVM bytecode,
built with Flex (lexer) and Bison (parser).

## Features
- Lexical analysis with Flex
- Syntax & semantic analysis with Bison
- Symbol table implementation
- Type checking
- JVM bytecode generation

## How to build
make

## How to run
./constraints <input_file>

## Requirements
- gcc
- flex
- bison

## Language
C
